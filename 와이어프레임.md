# 어린이 체육시설·강사·팀 매칭 플랫폼 — 화면 단위 와이어프레임 (초안)

작성일: 2026-07-21
범위: 6장 MVP 포함 기능 기준, 학부모용 화면만 다룸 (Admin 콘솔은 기술아키텍처.md 0절 참고, 별도 문서 대상 아님)
표기: 모바일 웹(반응형) 기준 세로 레이아웃

## 화면 목록 및 플로우 맵

```
00 스플래시/로그인
  └─ 01 홈 (진입 분기)
        ├─ (A) 02 추천설문 → 03 추천결과 ─┐
        └─ (B) 04 검색필터 ───────────────┼─→ 05 검색결과
                                          │        │
                                          │        ▼
                                          │   06 상세페이지 (시설/강사/클래스)
                                          │        │
                                          │        ▼
                                          │   07 예약신청
                                          │        │
                                          │        ▼
                                          │   08 예약상태(신청완료/확정알림)
                                          │
                                          └─→ 09 마이페이지 (자녀프로필/예약내역)
                                                   │
                                                   ▼
                                              10 리뷰작성 (예약 완료 건만)
```

## 00. 스플래시 / 로그인·회원가입 (휴대폰 인증)

```
┌───────────────────────────┐
│                           │
│        [서비스 로고]        │
│    아이 체육, 맞춰서 찾다     │
│                           │
│  ┌─────────────────────┐ │
│  │ 휴대폰 번호 입력        │ │
│  └─────────────────────┘ │
│  ┌─────────────────────┐ │
│  │   인증번호 받기 →      │ │
│  └─────────────────────┘ │
│                           │
│   가입 시 이용약관/개인정보    │
│   처리방침에 동의합니다 (13장) │
└───────────────────────────┘
```
- 인증번호 입력 화면(별도)으로 이어짐 → 성공 시 최초 가입자는 **자녀 프로필 등록**(이름/생년월일/성별)으로 강제 이동, 기존 사용자는 01 홈으로

## 01. 홈 (진입 분기 — 3장 플로우의 시작점)

```
┌───────────────────────────┐
│ ☰   OO매칭         [마이]  │
├───────────────────────────┤
│  안녕하세요, OO맘님 👋       │
│  민준(7세)의 운동을 찾아볼까요?│
│                           │
│ ┌───────────┐┌───────────┐│
│ │ 종목을      ││ 종목을      ││
│ │ 모르겠어요   ││ 이미 알아요  ││
│ │  (A)      ││  (B)      ││
│ │ 추천받기 → ││ 바로검색 → ││
│ └───────────┘└───────────┘│
│                           │
│  🔥 우리 동네 인기 클래스     │
│  ┌─────────────────────┐ │
│  │ [사진] 아이스하키 새싹반  │ │
│  │ 아이스웍스 역삼 · ★4.8  │ │
│  └─────────────────────┘ │
│  (가로 스크롤 카드 리스트)     │
├───────────────────────────┤
│ [홈] [검색] [예약] [마이]    │
└───────────────────────────┘
```
- 카드 A/B가 3장 다이어그램의 분기점. 하단 탭바로 언제든 검색/예약/마이페이지 접근 가능

## 02. 추천 설문 (경로 A) — 다단계 입력

```
┌───────────────────────────┐
│ ← 2/4                     │
│                           │
│  민준이는 평소에 어떤 편인가요?│
│                           │
│  ┌─────────────────────┐ │
│  │ ○ 에너지 넘치고 활동적  │ │
│  │ ○ 차분하고 집중력 좋음  │ │
│  │ ○ 사교적, 친구와 잘 어울림│
│  │ ○ 승부욕이 강한 편     │ │
│  └─────────────────────┘ │
│  (복수 선택 가능)            │
│                           │
│           [다음 →]         │
└───────────────────────────┘
```
- 4단계 구성: **① 나이(자동, 자녀프로필 연동) → ② 성향 → ③ 체력/관심사 → ④ 거주지(자동, 프로필 연동)**
- 4.1절 "추천 설문 → 검색 결과 연결" 요구사항에 따라 입력값은 기술아키텍처.md `POST /recommend`로 전송

## 03. 추천 결과

```
┌───────────────────────────┐
│  민준이에게 이런 운동을        │
│  추천해요                   │
│                           │
│  ┌─────────────────────┐ │
│  │ 🏒 아이스하키   [92%맞음]│ │
│  │ 활동적 · 팀워크 · 승부욕  │ │
│  │ 이 지역 클래스 12개 →   │ │
│  └─────────────────────┘ │
│  ┌─────────────────────┐ │
│  │ ⛸️ 피겨스케이팅 [78%맞음]│ │
│  │ 집중력 · 개인종목       │ │
│  │ 이 지역 클래스 5개 →    │ │
│  └─────────────────────┘ │
│                           │
│      [다른 종목도 볼래요]     │
└───────────────────────────┘
```
- 근거(추천 사유 태그)를 카드에 노출 — 4.1절 "추천 종목 리스트 제공(근거 포함)" 요구사항 반영
- 카드 클릭 시 해당 종목이 필터로 적용된 **05 검색결과**로 바로 진입

## 04. 검색 필터 (경로 B 진입점)

```
┌───────────────────────────┐
│ ←   검색                   │
├───────────────────────────┤
│ 종목    [아이스하키 ▾]      │
│ 지역    [강남구 ▾] 반경2km  │
│ 자녀연령 [7세 ▾]            │
│ 요일    [월][화][수]...    │
│ 시간대   [평일저녁][주말]    │
│ 가격대   ────●──── 5~15만  │
│ 형태    [개인][그룹][팀]    │
│                           │
│         [검색결과 보기]      │
└───────────────────────────┘
```
- 4.1절 필터 축(종목/지역/연령/요일·시간/가격/형태) 그대로 반영. 지역은 기본값으로 프로필 주소 기반 자동 설정

- 결제는 현장/계좌이체임을 예약 전 명확히 고지 (10장 확정사항 — 기대와 실제 불일치 방지)

## 컴포넌트 재사용 정리

| 컴포넌트 | 사용 화면 |
|---|---|
| 클래스 카드 (사진/이름/평점/거리/가격/빈자리) | 01(인기클래스), 05(검색결과), 03(추천결과 연결) |
| 예약 상태 배지 (신청중/확정/완료/취소) | 08, 09 |
| 자격 인증 배지 | 06 상세페이지 |
| 자녀 선택 드롭다운 | 02, 07 |

## MVP 제외 화면 (2차 확장, 6장 기준)

- 강사/시설용 운영 대시보드(출석관리, 정산) → 공급자 셀프 온보딩 전환(12.3절) 시점에 별도 설계
- 실시간 채팅 (학부모↔강사/시설)
- 온라인 결제/보증금 입력 화면 (11.1절 PG 연동 시점에 07 예약신청 화면에 결제 단계 추가)
